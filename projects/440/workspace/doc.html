Here we are testing some ideas about using COMBINE Archive Manifest files to define the relevant resources in a workspace and then adding annotations to describe the linkage between entities in the SVG image and the CellML models and SED-ML experiments. We will begin with the RDF annotations being in this workspace (and listed in the manifest), but in the general case tools would query PMR for any relevant annotations.

The SED-ML description of the main simulation experiment is available; as is the main CellML model defining this particular stimulus protocol and the base mathematical model.
