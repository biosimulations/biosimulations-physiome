From this page you can access the CellML model and various other information from the "Views available" menu on the right. This CellML 1.0 version of the S1 simulation experiment was generated from the CellML 1.1 version using Jonathan Cooper's VersionConverter tool. This CellML 1.0 version is provided as a convenience for use in tools which do not support the normative CellML 1.1 version of this model. This generated version of the model loads and runs fine in OpenCell but not COR due to the use of the rem and min MathML operators. There may also be other reasons why this model does not run in COR.
